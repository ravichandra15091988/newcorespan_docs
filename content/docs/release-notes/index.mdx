### K8s Config

```bash
sed -i 's/k8_ha_enabled: true/k8_ha_enabled: false/g' ~/dsp/vars/k8s.yaml
sed -i 's/#   - hostname: "DSP-kube-master1"/    - hostname: "FMmanager"/' ~/dsp/vars/k8s.yaml
sed -i '/- hostname: "FMmanager"/{n;s/.*/      ansible_group: masters/}' ~/dsp/vars/k8s.yaml
sed -i 's/#   - hostname: "DSP-kube-worker1"/    - hostname: "FMworker1"/' ~/dsp/vars/k8s.yaml
sed -i '/- hostname: "FMworker1"/{n;s/.*/      ansible_group: workers/}' ~/dsp/vars/k8s.yaml
sed -i 's/#   - hostname: "DSP-kube-worker2"/    - hostname: "FMworker2"/' ~/dsp/vars/k8s.yaml
sed -i '/- hostname: "FMworker2"/{n;s/.*/      ansible_group: workers/}' ~/dsp/vars/k8s.yaml
sed -i 's/# k8s_machines:/	k8s_machines:/' ~/dsp/vars/k8s.yaml
```
